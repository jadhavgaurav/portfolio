# Cinematic AI Portfolio

A high-performance, immersive portfolio website built with **Next.js 14 (App Router)**, **Tailwind CSS**, and **Framer Motion**. Designed to showcase AI engineering projects with cinematic visual fidelity.

## âœ¨ Features

- **Cinematic Visuals:** Ambient backgrounds, particle effects, and holographic UI elements.
- **Interactive UI:** Custom cursor, magnetic buttons, and 3D tilt effects on project cards.
- **Performance:** Server-Side Rendering (SSR) for initial load, optimized fonts, and minimal client-side hydration.
- **Responsive:** Fully responsive design adapting to all device sizes.
- **Clean Code:** Modular component architecture with strict TypeScript typing.
